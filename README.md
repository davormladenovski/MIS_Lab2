# Meal App

Flutter апликација за прегледување на рецепти од TheMealDB API.

## Функционалности

- **Листа од категории** - Преглед на сите категории на јадења со слика и опис
- **Пребарување на категории** - Брзо наоѓање на категории
- **Јадења по категорија** - Приказ на сите јадења од избрана категорија во grid layout
- **Пребарување на јадења** - Пребарување на јадења од избраната категорија
- **Детален рецепт** - Целосен приказ на рецепт со состојки, инструкции и YouTube линк
- **Рандом рецепт** - Приказ на случаен рецепт на денот

## Структура на проект

```
lib/
├── models/          # Модели за Category, Meal, Recipe
├── screens/         # Екрани на апликацијата
├── services/        # API сервис за TheMealDB
└── widgets/         # Пресметани widgets
```

## Инсталација

1. Клонирајте го проектот
2. Инсталирајте ги зависностите:
   ```bash
   flutter pub get
   ```
3. Стартувајте ја апликацијата:
   ```bash
   flutter run
   ```

## Зависности

- `http` - За HTTP барања до API
- `url_launcher` - За отворање на YouTube линкови
